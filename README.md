# 온도에 따른 옷 추천 웹사이트

- 거주지를 검색하여 그날의 기온에 입어야 할 옷을 추천해 주는 웹사이트
- MVC 모델을 사용.

## 추가해야하는 기능 :

- 거주지 데이터베이스 - 100%
- 거주지 Entity - 100%
- 거주지 DAO - 90%
- 기상 API - 100%
- (기상 API)요청받은 결과값 정리 - 90%
- HTML, JS, CSS - 70%
- 거주지 목록 AutoComplete - 100%
- 현재 위치 - 0%
- Controller - 50%
